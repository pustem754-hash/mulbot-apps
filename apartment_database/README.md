# База квартир

Система управления базой данных квартир с веб-интерфейсом.

## Возможности

- Добавление новых квартир
- Редактирование существующих записей
- Удаление квартир
- Поиск и фильтрация по параметрам
- Управление статусами (доступна/продана/забронирована)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу: http://localhost:5000

## Структура данных

Каждая квартира содержит следующие поля:
- Адрес
- Количество комнат
- Площадь (м²)
- Этаж и общее количество этажей
- Цена
- Статус (доступна/продана/забронирована)
- Описание
- Данные владельца (имя и телефон)

## Технологии

- Python 3.x
- Flask (веб-фреймворк)
- SQLAlchemy (ORM)
- SQLite (база данных)
- Bootstrap (интерфейс)
- WTForms (формы)
